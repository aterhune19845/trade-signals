## What is published (and what is not)

- **Published:** daily target weights, exposures, and a normalized performance
  index (start = 100).
- **Never published:** account balances, dollar amounts, broker details, API
  keys, or any credential.

## How to read the signals

`signals/history.jsonl` is one JSON object per line, appended once per trading
day. Fields: `published_utc`, `data_date`, `weights` (tradeable ETF tickers →
target fraction), `cash`, `eq_exposure`, `gold_exposure`, `cat_halted`, `engine`.
